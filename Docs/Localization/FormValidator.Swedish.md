Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in Swedish.

### Authors

* Martin Lundgren

### Usage

Simply change the language property to *sv-SE*.

	MooTools.lang.setLanguage("sv-SE");
