Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in Danish.

### Authors

* Martin Overgaard

### Usage

Simply change the language property to *da-DK*.

	MooTools.lang.setLanguage("da-DK");
