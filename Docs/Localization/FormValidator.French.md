Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in French.

### Authors

* Miquel Hudin

### Usage

Simply change the language property to *FRA*.

	MooTools.lang.setLanguage("FRA");

See also: [FormValidator][] and [MooTools.Lang][]
