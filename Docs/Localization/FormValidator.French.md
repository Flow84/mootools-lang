Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in French.

### Authors

* Miquel Hudin

### Usage

Simply change the language property to *fr-FR*.

	MooTools.lang.setLanguage("fr-FR");
