Class: Date {#Date}
=====================================

Contains the messages for [Date.Extras][] in English specific to the UK.

### Authors

* Aaron Newton

### Usage

Simply change the language property to *en-GB*.

	MooTools.lang.setLanguage("en-GB");

See also: [Date.Extras][] and [MooTools.Lang][]
