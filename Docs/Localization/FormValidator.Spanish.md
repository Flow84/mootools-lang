Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in Spanish.

### Authors

* Miquel Hudin

### Usage

Simply change the language property to *es-ES*.

	MooTools.lang.setLanguage("es-ES");

See also: [FormValidator][] and [MooTools.Lang][]
