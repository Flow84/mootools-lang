Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in German (Switzerland).

### Authors

* Michael van der Weg

### Usage

Simply change the language property to *de-CH*.

	MooTools.lang.setLanguage("de-CH");
