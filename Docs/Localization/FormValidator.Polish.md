Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in Polish.

### Authors

* Oskar Krawczyk

### Usage

Simply change the language property to *pl-PL*.

	MooTools.lang.setLanguage("pl-PL");

See also: [FormValidator][] and [MooTools.Lang][]
