Class: FormValidator {#FormValidator}
=====================================

Contains the messages for [FormValidator][] in German.

### Authors

* Frank Rossi
* Ulrich Petri

### Usage

Simply change the language property to *de-DE*.

	MooTools.lang.setLanguage("de-DE");
